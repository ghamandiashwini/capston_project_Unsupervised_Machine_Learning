# Unsupervised Learning Analysis

## Overview

The **Unsupervised Learning Analysis** project was completed as part of the Data Science & AI Bootcamp. The objective was to apply unsupervised machine learning techniques to explore unlabeled data, identify hidden patterns, and group similar observations using clustering and dimensionality reduction methods.

The project demonstrates the end-to-end workflow of data preprocessing, feature scaling, clustering, Principal Component Analysis (PCA), and data visualization.

---

## Objectives

- Explore and understand the dataset.
- Prepare and preprocess data for unsupervised learning.
- Apply clustering techniques to identify natural groupings.
- Use Principal Component Analysis (PCA) to reduce dimensionality.
- Visualize clusters and interpret the results.
- Generate insights from unlabeled data.

## Project Workflow

### 1. Data Exploration
- Dataset inspection
- Descriptive statistics
- Feature analysis
- Missing value identification

### 2. Data Preprocessing
- Data cleaning
- Feature scaling using StandardScaler
- Preparing data for clustering

### 3. Clustering
- K-Means Clustering
- Determining the optimal number of clusters using the Elbow Method
- Cluster assignment and visualization

### 4. Dimensionality Reduction
- Principal Component Analysis (PCA)
- Two-dimensional visualization of clustered data
- Interpretation of principal components

## How to Run

1. Clone the repository

```bash
git clone https://github.com/ghamandiashwini/capston_project_Unsupervised_Machine_Learning.git
```

2. Navigate to the project folder

```bash
cd unsupervised_task
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open the notebook and run all cells.
